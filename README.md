Real-Time Private Messenger
===========================

A full-stack, real-time chat application designed for secure private communication. This application utilizes WebSockets for instant message delivery and provides a robust contact invitation system.

✨ Features
----------

*   **Real-Time Messaging:** Instant communication powered by Socket.io.
    
*   **User Authentication:** Secure login and registration system.
    
*   **Contact/Invitation System:** Send and accept friend requests to start private chats.
    
*   **Online Status:** Real-time visibility of who is currently online.
    
*   **Profile Management:** Update user credentials easily.
    
*   **Unread Indicators:** Notification badges for unread messages.
    

🛠 Tech Stack
-------------

*   **Frontend:** React (Vite), Socket.io-client
    
*   **Backend:** Node.js, Express, Socket.io
    
*   **Database:** MongoDB (via MongoDB Atlas)
    
*   **Deployment:** Vercel (Frontend), Render (Backend)
    

⚙️ Setup Instructions (Local Development)
-----------------------------------------

### Prerequisites

*   Node.js (v18+)
    
*   MongoDB Atlas Account (or a local MongoDB instance)
    

### 1\. Clone the Repository

1.  Bash git clone \[Your Repository Link\]
    
2.  cd chat-assessment
    

### 2\. Backend Setup

1.  Bash cd backend
    
2.  Code snippetPORT=5000MONGO\_URI=your\_mongodb\_connection\_string
    
3.  Bash npm install npm start
    

### 3\. Frontend Setup

1.  Bash cd frontend
    
2.  Bash npm install
    
3.  Bash npm run dev
    
4.  Open the provided local URL (e.g., http://localhost:5173) in your browser.
